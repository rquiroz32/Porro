# Porro

# Description
This is an application that displays the weather for a user searched city, displays an icon to illustrate the type of weather, and a forecast for the next 5 days.

# Technologies Used:
- HTML
- CSS
- BootStrap CSS
- JavaScript
- JQuery
- moment.js
- Open Weather Map API
